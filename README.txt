== Description ==

Unlock the full potential of AI-powered content generation to craft engaging, high-quality content that aligns with your brand's identity. GPT SEO and Content Writer Assistant makes it easier than ever to create captivating titles, headings, paragraphs, and even complete blog posts in just a few clicks.

### Features

* Simple setup: Connect your OpenAI API key and customize your brand settings in minutes.
* Personalized content: Tailor your content's tone and style to match your brand's voice.
* Versatile content generation: Create titles, headings, paragraphs, and full-length blog posts with ease.
* Meta box integration: Generate content directly within the WordPress editor for any post type.
* SEO-optimized content: Boost your site's visibility and search engine ranking with AI-generated content designed for SEO.

== Frequently Asked Questions ==

= What is GPT SEO and Content Writer Assistant? =

GPT SEO and Content Writer Assistant is a WordPress plugin that integrates GPT technology to help users create SEO-optimized and engaging content for their websites. It generates personalized titles, headings, paragraphs, and blog posts that align with your brand's voice.

= How do I set up the plugin? =

After installing the plugin, connect your OpenAI API key and customize your brand settings, including your business name, description, and desired tone. This simple setup process ensures your generated content aligns with your brand identity.

= Can I use the plugin for different post types? =

Yes, the plugin integrates with all post types in WordPress, including pages, posts, and custom post types. You can generate content directly within the WordPress editor via a meta box.

= How do I generate content using the plugin? =

Once you have set up the plugin, navigate to a page in the dasboard and you'll see the GPT Assistant meta box on the side. Choose the type of content you want to generate (e.g., title, heading, paragraph, or blog post) and provide a short description. The plugin will then generate content based on your preferences and input.

= Is the content generated SEO-optimized? = 

Yes, the plugin is designed to create content that is optimized for search engines, helping to improve your website's visibility and ranking.

= How is my OpenAI API Key used? =

Your API key is securely stored in your WordPress database and is sent directly to the Open AI API. We do not send your API Key through any third party services.
